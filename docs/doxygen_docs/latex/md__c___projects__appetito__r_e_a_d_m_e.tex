Restaurant order management system based on C++, with one multithreaded server and two clients, networking implemented using TCP and sockets with Asio library, serialization implemented using Cereal library. The goal is to be as efficient, modular, and deployable as we are able to implement. Work in progress. This is a student project by three students at Hefei University of Technology.\hypertarget{md__c___projects__appetito__r_e_a_d_m_e_autotoc_md49}{}\doxysection{Roadmap}\label{md__c___projects__appetito__r_e_a_d_m_e_autotoc_md49}
